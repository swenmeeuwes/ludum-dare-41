Folder structure

│   
├───Animations
│       Contains animation controllers and animations for the game
│       
├───Editor
│       Contains scripts that override the default unity editor
│       
├───Materials
│       Contains material objects
│       
├───Models
│       Contains 3d models, preferably blender files
│       
├───Plugins
│       Contains plugins from the asset store
│       
├───Prefabs
│       Contains game objects that have been made into a prefab
│       
├───Resources
│       Contains resources that will be loaded dynamically in game
│       
├───Scenes
│   │   Contains all game scenes  
│   │   
│   └───Playground
│           Contains all test scenes, here you can play around with new mechanics
│           
├───Scripts
│       Contains all game scripts
│       
├───Sprites
│       Contains 2d image, mainly used for UI components
│
├───Shaders
│       Contains all shader files
│       
└───Textures
        Contains textures that can be applied to terrain and environment
